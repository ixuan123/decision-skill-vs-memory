# decision-skill-vs-memory

> 每次完成任务后，自动判断这个经验该存哪。

---

## 这是什么？

一个决策规则。每次我干完一件事，它会问我：

> "这件事，值得记下来下次直接用吗？"

如果是 → 存成 **Skill**（技能）
如果不是 → 存成 **Memory**（记忆）

就这么简单。

---

## 为什么要判断？

不判断的话会出两种问题：

1. **什么都记** → 脑子越来越乱，找东西找不到
2. **什么都不记** → 每次都要重新学，越干越笨

---

## 判断标准

就一条：

> **这件事，下次还会用吗？**

- 下次还会用 → Skill ✅
- 下次不会用 → Memory 📝

就这么一条，不用多想。

---

## 例子

| 事情 | 下次还用吗？ | 存哪 |
|------|-------------|------|
| 配置 API Key | 不用，配一次就行 | Memory |
| Chrome 读抖音 | 用，经常干 | Skill |
| 删除重复 skill | 不用，一次性的 | Memory |
| 横纵分析法 | 用，分析都用 | Skill |
| 父亲大人不让重启 | 不用，一条规矩 | Memory |

---

## 记住一句话

**下次还用就存 Skill，不用就存 Memory。**

---
